# Android Encryption:
* Android v3 - Mostly no encryption
* Android v7 - File disk encryption
* Android v10 - File Based encryption

## Full Disk Encryption (FDE)
- Encryption key is applied to the entire disk
- Access to user data is decrypted after boot up
- Screencode is involved in decryption of disk
- Factory installed Android 9 or below

## File Based Encryption (FBE)
- Encryption key is unique to per file
- User data is decrypted after unlock
- Screen passcode used for decryption of contents
- Android v10 and above
